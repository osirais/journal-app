# journal-app

## Web

To run the web version of **journal-app** (built with Next.js):

```sh
cd web
bun i
bun d
```

## Mobile App

To run the mobile version of **journal-app** (built with Expo):

```sh
cd app
bun i
bun start
```

Then scan the QR code on your terminal using the Expo Go app on your phone.
